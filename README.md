# Language Modeling Predictor (Unigram/Bigram/Uniform)

## Overview
Built unigram, bigram, and uniform language models to analyze folklore corpora based on the works of authors, Andersen and the Grimm Brothers.
Compared vocabulary usage and generated sample text using probability distributions.

## Tech Stack
Python, NumPy, Matplotlib

## Project Details
- Cleans + tokenizes text corpora
- Computes unigram and bigram probabilities
- Finds high-frequency words and compares corpora
- Generates new text samples from learned distributions
- Visualizes vocabulary patterns

## How to run
```bash
python3 language_modeling_predictor.py
